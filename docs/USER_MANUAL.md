## 1. 시스템 개요

학교 전산실 워크스테이션(Linux ubuntu 환경)을 **브라우저에서 직접 사용**할 수 있는 시스템입니다. 별도 프로그램 설치 없이 학교 내·외부 어디서나 접속 가능합니다.

- 지원 브라우저: Chrome, Firefox, Edge (최신 버전 권장)
- 필요 사양: 인터넷 연결만 있으면 됩니다
- VPN 불필요: Cloudflare 고정 도메인 사용

---

## 2. 접속 및 로그인

1. 브라우저에서 포털 주소 접속
2. **Google로 로그인** 버튼 클릭
3. 학교 Google 계정(`@ts.hs.kr` 등)으로 인증
4. 로그인 성공 시 대시보드로 자동 이동

> 개인 Google 계정이 아닌 **학교 계정**으로 로그인해야 합니다. 접근이 안 될 경우 관리자에게 문의하세요.

## 3. PC 대여하기

### 3-1. 사용 가능한 PC 확인

대시보드에 전체 서버 목록이 표시됩니다.

| 상태 표시 | 의미 |
|---|---|
| **사용 가능** | 바로 대여 가능 |
| **1명 사용 중** | 1명 사용 중, 추가 1명 가능 |
| **가득 참** | 현재 만석 (다른 PC 선택) |
| **오프라인** | 서버 점검 중 |

각 서버 카드에는 CPU, RAM, GPU 정보가 표시됩니다. GPU가 필요한 작업(딥러닝, 그래픽 등)은 RTX 3080 또는 RTX 2080 Ti 서버를 선택하세요.

### 3-2. 세션 시작

1. 사용할 서버 카드의 **PC 대여하기** 버튼 클릭
2. 잠시 기다리면 (약 30초~1분) 데스크탑 화면이 나타남
3. 화면이 나타나면 바로 사용 가능

> **첫 부팅 시 약 30~60초** 소요됩니다. 검은 화면이 보이면 잠시 기다려 주세요.

## 4. 데스크탑 사용하기

### 4-1. 기본 조작

- 마우스/키보드: 일반 PC와 동일하게 사용
- 파일 관리자: 작업 표시줄 하단 아이콘 클릭 또는 바탕화면 우클릭
- 기본 제공 프로그램: Chrome, Firefox, VS Code

### 4-2. 한국어 입력

1. **Shift + Space** 키로 한/영 전환
2. 화면 우측 하단 트레이에서 현재 입력 모드 확인 가능
3. 세션 시작 후 약 **5~8초** 후 한국어 입력기가 활성화됩니다

> 세션 시작 직후에는 한국어 입력이 안 될 수 있습니다. 잠시 기다린 후 사용하세요.

### 4-3. 클립보드 사용 (내 PC ↔ 서버)

KasmVNC 좌측 탭 메뉴 → **클립보드** 아이콘을 통해 텍스트 복사/붙여넣기 가능합니다.

### 4-4. 파일 업로드/다운로드

**업로드 (내 PC → 서버)**

포털 대시보드의 **파일 업로드** 버튼을 사용합니다.

1. 대시보드에서 **파일 업로드** 버튼 클릭
2. 업로드할 파일 선택
3. 업로드 완료 후 서버 데스크탑의 `~/Downloads` 폴더에서 확인

> 학교 내부망에서 접속 시 LAN 직접 전송으로 속도가 빠릅니다. 외부 접속 시에는 Cloudflare 터널을 경유합니다.

**다운로드 (서버 → 내 PC)**

- KasmVNC 좌측 탭 메뉴 → **파일 관리자** 아이콘 사용
- 또는 서버 내 브라우저(Chrome/Firefox)로 직접 다운로드 후 클립보드 활용

## 5. 세션 일시 중지 (이어하기 기능)

작업 중간에 나갔다가 나중에 이어서 사용할 수 있습니다.(최대 한 달간 유지) 세션 종료 버튼을 누르거나 유지 기간이 지나기 전까지는 크롬 탭을 닫기만 해서 세션이 종료되지는 않습니다.

### 일시 중지

1. 대시보드로 이동
2. **세션 종료** 버튼 클릭 → 컨테이너가 일시 중지됩니다 (파일·설치 프로그램 보존)

### 재개

다시 로그인하면 대시보드에 이전 세션이 표시됩니다.

- **이어서 사용하기**: 기존 작업 상태 그대로 재개. 컨테이너가 원래 서버에서 그대로 재시작되므로 설치된 프로그램과 파일이 보존됩니다.
- **새로 시작하기**: 기존 컨테이너 삭제 후 새 세션 시작. 이 경우 **원래 PC와 다른 서버**를 선택해서 시작할 수 있습니다.

> **이어서 사용하기**는 브라우저(내 노트북, 학교 PC 등)가 달라도 동일한 서버 환경으로 재개됩니다.

> **새로 시작하기**로 다른 서버를 선택하면 이전 서버의 파일·설치 프로그램은 삭제됩니다. 중요한 파일은 미리 다운로드해 두세요.

---

## 6. 세션 영구 종료

작업이 완전히 끝났다면:

1. 대시보드 → **영구 삭제** 버튼 클릭
2. 확인 후 컨테이너가 완전히 삭제됩니다

> 영구 종료 시 서버에 저장한 파일은 모두 삭제됩니다. 중요한 파일은 미리 다운로드하세요.

---

## 7. 주의사항

- **한 사용자당 최대 1개 세션**을 동시 사용 가능합니다
- **서버당 최대 2명** 동시 사용 가능합니다 (자원 자동 분배)
- 데스크탑 안에서 **재부팅/종료 명령**을 실행해도 자동으로 재시작됩니다
- 장시간 미사용 시 관리자에 의해 세션이 회수될 수 있습니다

## 8. 자주 묻는 질문 (FAQ)

**Q. 화면이 검은색으로만 보입니다.**  
A. 세션 시작 중입니다. 약 30~60초 기다린 후 새로고침 해보세요.

**Q. 한국어 입력이 안 됩니다.**  
A. 세션 시작 후 5~8초 기다린 후 Shift+Space를 눌러보세요. 접속 컴퓨터의 키보드는 영어로 변경 후 Shift+Space를 눌러야 자소 분리가 안 됩니다.

**Q. 서버가 전부 "가득 참"으로 표시됩니다.**  
A. 현재 모든 PC가 사용 중입니다. 잠시 후 다시 시도하거나 관리자에게 문의하세요.

**Q. 작업하던 파일이 사라졌습니다.**  
A. "새로 시작하기"로 세션을 시작했거나 영구 삭제가 된 경우 복구가 어렵습니다. 중요 파일은 항상 로컬에 다운로드해 두세요.

**Q. 속도가 너무 느립니다.**  
A. 브라우저 화질 설정을 낮추거나, 다른 서버를 선택해 보세요.
